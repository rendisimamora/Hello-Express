# Hello Express

# How to starter this application?

1. Clone the repository.
```
$ git clone https://github.com/rendisimamora/Hello-Express.git
```

2. Go to repo in local.
```
$ cd hello-express
```

3. Install node environment.
```
$ npm install
```

4. Running the application.
```
$ npm run dev
```
